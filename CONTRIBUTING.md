# Contributing to Awesome Cloud Native

Thanks for your interest in contributing! This list is curated by [Podo Stack](https://podostack.substack.com), but we welcome suggestions from the community.

## How to suggest a tool

1. **Open an issue** using the [suggest tool template](https://github.com/iGusev/awesome-cloud-native/issues/new?template=suggest-tool.md)
2. Provide the required information:
   - Tool name and GitHub link
   - One-line description (what it does)
   - Why it belongs here (production-readiness, unique value)

## What we don't include

- Paid-only tools (open source core required)
- Abandoned projects (no commits in 12+ months)
- Tools that duplicate existing entries without clear differentiation
- Self-promotion without substance

## Pull Requests

If you want to submit a PR directly:

1. Fork the repository
2. Add your tool in the appropriate category
3. Follow the format above
4. Submit a PR with a clear description

We'll review and merge if it fits our criteria.
